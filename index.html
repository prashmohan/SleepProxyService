SleepProxyService by prashmohan

Saving the world one cluster at a time

Sleep Proxy Service is a research prototype pursued at the University of California at Berkeley. The scope of the project is to manage the nodes in a cluster going to sleep and coming back alive. This is done in order to save aggregate energy used in the cluster.

Dependencies

Ganglia (gmond, gmetad--python)

Install

Master Node: Put sleep_proxy_plugin.py and common.py inside your gmetad-python plugin directory and start gmetad.py. Also install CCexec.py into all of these nodes. Slave Nodes: Put CCd.py and common.py inside a common directory (say CCd) and run CCd.py on all nodes (make sure gmond is running before this).
